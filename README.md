# proxmox-management

Tools and automation to provision and manage Proxmox VE infrastructure. This repository will contain playbooks, scripts and documentation to help provision VMs, manage backups, and configure networking on Proxmox clusters.

Repository layout (initial):
- README.md - project overview
- LICENSE - project license (MIT)
- .gitignore - common ignores
- playbooks/site.yml - placeholder Ansible playbook to be expanded

Getting started
1. Clone the repository:
   git clone https://github.com/tk-4/proxmox-management.git
2. Add playbooks and scripts in the `playbooks/` directory.
